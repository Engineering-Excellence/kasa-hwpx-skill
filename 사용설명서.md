# 쉬운 사용설명서 📘

> 컴퓨터가 익숙하지 않아도 괜찮습니다.
> 이 도구는 **어려운 명령어 없이, 평소 말하듯 부탁하면** 우주항공청 양식의 한글 보고서를 만들어 줍니다.
> 천천히 따라오시면 됩니다.

---

## 1. 이게 뭐 하는 물건인가요?

빈칸이 정해진 **결재 서류 양식**을 떠올려 보세요.
보통은 사람이 그 양식을 열어 제목·날짜·표·내용을 일일이 채웁니다.

이 도구(`kasa-hwpx`)는 그 일을 **대신 해 주는 비서**입니다.
당신이 "이런 내용으로 보고서 하나 만들어 줘"라고 부탁하면,
우주항공청 표준양식(표지·로고·여백)은 그대로 둔 채 **내용만 알아서 채워**
완성된 한글 파일(`.hwpx`)을 돌려줍니다.

쉽게 말해 — **말로 시키면 보고서가 나옵니다.**

---

## 2. 무엇을 해 줄 수 있나요?

크게 여섯 가지를 합니다.

1. **새 보고서 만들기**
   제목과 내용만 알려 주면 우주항공청 양식에 맞춰 새 보고서를 만들어 줍니다.

2. **있던 보고서 고치기 (재기안)**
   작년 보고서처럼 이미 있는 한글 파일을 주면,
   **모양은 그대로 두고 숫자·날짜·부서 이름 같은 글자만** 바꿔 줍니다.
   ("2025년 → 2026년"처럼요.)

3. **검사하기**
   만든 보고서가 양식 규정에 맞는지, 글자가 겹치지 않는지 **자동으로 점검**해 줍니다.
   날짜를 `2026. 7. 7.`처럼 쓰는 **공문서 표기법**(날짜·시간·숫자 쉼표·항목 기호)까지
   맞았는지 함께 봐 줍니다.

4. **직인·서명 이미지 넣기** ✨새 기능
   보고서의 원하는 자리(예: "(직인)"이라고 적힌 곳)에 **도장 그림이나 서명 이미지**를
   넣어 줍니다. 이미 들어간 도장을 새 도장으로 바꾸거나 지우는 것도 됩니다.

5. **꼬리말·쪽번호·표 고치기**
   완성된 파일의 꼬리말(맨 아래 글), 쪽번호, 표의 칸·줄을 모양은 지키면서 고쳐 줍니다.

6. **개인정보 안전하게 채우기**
   이름·연락처 같은 개인정보가 들어가는 양식은, 그 값이 **화면이나 기록에 남지 않게**
   채우는 특별한 방법(secure-fill)을 씁니다.

> 이 밖에도 이런 것들을 할 줄 압니다:
> - 다른 프로그램에서 변환하다 **글자가 세로로 깨진 파일** 고치기 ("글자가 세로로 나와요")
> - **두 보고서 비교(신구대조)** — 지난번 파일과 이번 파일에서 **무엇이 바뀌었는지**
>   문단·표 칸 단위로 찾아 줍니다 ("이 두 파일 뭐가 달라졌는지 비교해 줘")
> - **빈칸 양식 채우기** — "성명:" 옆처럼 라벨 옆 **빈 칸을 찾아** 값을 넣어 줍니다
>   ("이 양식에 성명은 홍길동, 소속은 우주수송정책과로 채워 줘")

## 3. 시작하기 전에 (준비물)

- **Claude 또는 Cowork 앱** — 여기에 대고 부탁하면 됩니다.
- **이 도구(스킬)를 앱에 한 번 등록** — 보통 **처음 한 번만** 하면 됩니다.
  설치가 어렵게 느껴지면, 주변의 전산 담당자나 개발자에게
  "이 깃허브 스킬을 등록해 달라"고 부탁하세요. 한 번만 해 두면 그다음부터는 당신이 말로 쓰면 됩니다.
- **한글(한컴오피스) 프로그램** — 완성된 `.hwpx` 파일을 열어 보고 인쇄할 때 필요합니다.

> 💡 표지 로고까지 완벽하게 보려면 **한글 프로그램에서 여는 것**이 가장 정확합니다.

## 4. 가장 쉬운 사용법 — 말로 시키기

설치가 끝났다면, 이제 **명령어를 외울 필요가 전혀 없습니다.**
앱 채팅창에 평소처럼 한국어로 부탁하면 됩니다.

### (1) 새 보고서 만들기

이렇게 말해 보세요 👇

```
우주항공청 양식으로 보고서 하나 만들어 줘.
제목: 누리호 5차 발사 대비 국내여비 추가 확보 요청
작성: 2026년 6월, 우주수송정책과
내용:
- 요청 개요: 5차 발사 준비로 출장이 늘어 국내여비가 부족함
- 집행 현황: 연간 예산 2,800만 원 중 상반기 2,100만 원(75%) 집행
- 요청 사항: 1,100만 원 추가 확보 요청
```

이렇게 **제목 / 작성정보(날짜·부서) / 들어갈 내용**만 알려 주면,
나머지(□ㅇ- 같은 기호 위계, 표, 줄 맞춤)는 **알아서** 정리해 줍니다.

> 💡 정보가 빠지면 되물어봅니다. 예: "작성 부서가 어디인가요?"
> 편하게 답하시면 됩니다. 한 번에 완벽하게 적지 않아도 괜찮습니다.

### (2) 있던 보고서 고치기 (재기안)

작년 파일이나 비슷한 보고서가 이미 있다면, 그 파일을 올리고 이렇게 말하세요 👇

```
이 보고서에서 '2025년'은 '2026년'으로,
'기획팀'은 '우주수송정책과'로 바꿔서 다시 만들어 줘.
모양은 그대로 두고 글자만 바꿔 줘.
```

표·여백·글꼴은 **하나도 건드리지 않고**, 말한 글자만 바꿔서 새 파일을 줍니다.

### (3) 검사 부탁하기

```
방금 만든 보고서가 양식에 맞는지 검사해 줘.
```

문제가 있으면 어디가 잘못됐는지 알려 줍니다.
양식뿐 아니라 **공문서 표기법**도 봐 줍니다 — 예를 들어 "2026-07-07"이라고 썼으면
"공문서에서는 `2026. 7. 7.`로 씁니다"라고 짚어 줍니다.

### (4) 직인·서명 이미지 넣기

도장 그림 파일(예: `직인.png`)을 올리고 이렇게 말하세요 👇

```
이 보고서의 "(직인)" 자리에 올려 준 도장 이미지를 2cm 크기로 넣어 줘.
```

"(직인)" 글자가 지워지고 그 자리에 도장 그림이 들어간 새 파일을 돌려줍니다.
도장을 바꾸고 싶으면 "이 도장을 새 도장으로 바꿔 줘", 지우고 싶으면
"이 도장을 지워 줘"라고 하면 됩니다.

> 💡 표지의 우주항공청 로고는 실수로 지워지지 않게 **보호**되어 있습니다.

### (5) 두 보고서 비교하기 (신구대조)

지난번 파일과 이번 파일을 둘 다 올리고 이렇게 말하세요 👇

```
이 두 보고서에서 뭐가 바뀌었는지 비교해 줘.
```

"추가 1건 · 수정 2건 · 동일 40건"처럼 요약하고, 바뀐 문단과 **표의 바뀐 칸**까지
"88,000 → 92,000"처럼 짚어 줍니다. 원본 파일은 건드리지 않습니다.

## 5. 완성된 파일 여는 법

1. 받은 파일 이름이 `○○○.hwpx`로 끝나는지 확인합니다. (이게 한글 파일입니다.)
2. 그 파일을 **두 번 클릭**하면 한글(한컴오피스) 프로그램에서 열립니다.
3. 표지·로고·본문이 보이면 성공입니다. 그대로 인쇄하거나 수정하면 됩니다.

> 파일이 한글이 아닌 다른 프로그램에서 열리려 한다면,
> 한글 프로그램을 먼저 켠 뒤 **파일 → 열기**로 불러오세요.

## 6. (조금 익숙해지면) 직접 명령어로 쓰는 법

컴퓨터에 조금 익숙하고 "내 손으로 직접 돌려보고 싶다" 하시면,
아래 **네 줄**만 알면 됩니다. (검은 명령창에 한 줄씩 입력)

```bash
# ① 새 보고서 만들기  (spec.json = 내용을 적어 둔 파일)
python3 kasa-hwpx/scripts/build_report.py --spec spec.json --output 보고서.hwpx

# ② 있던 보고서 글자만 바꾸기  (repl.json = 바꿀 단어 목록)
python3 kasa-hwpx/scripts/redraft.py --input 원본.hwpx --map repl.json --output 결과.hwpx

# ③ 검사하기 (양식 + 공문서 표기법)
python3 kasa-hwpx/scripts/validate.py 보고서.hwpx --kasa

# ④ "(직인)" 자리에 도장 이미지 넣기 (20mm 크기)
python3 kasa-hwpx/scripts/hwpx_image.py add 보고서.hwpx --image 직인.png \
        --anchor "(직인)" --replace-anchor --width-mm 20 --output 결과.hwpx
```

> `spec.json`·`repl.json`을 직접 쓰는 게 어렵다면, 그냥 4번처럼 **말로 시키는 방법**을 쓰세요.
> 결과는 똑같습니다.

## 7. 자주 묻는 질문

**Q. 한글 프로그램이 꼭 있어야 하나요?**
A. 보고서를 **만드는 것**은 한글 없이도 됩니다. 다만 완성본을 **열어서 보거나 인쇄**하려면 한글이 필요합니다.

**Q. 제가 만든 내용이 인터넷에 올라가나요?**
A. 아니요. 이 도구는 당신의 컴퓨터(또는 앱 안)에서 양식을 채울 뿐입니다.

**Q. 표지 로고나 여백이 바뀌지 않을까요?**
A. 바뀌지 않습니다. 표준양식의 표지·로고·여백은 그대로 두고 **본문 글자만** 채웁니다.

**Q. 글자가 겹쳐 보여요.**
A. "검사해 줘"라고 하면 자동으로 잡아 줍니다. 이 도구는 줄 겹침을 막는 처리가 기본으로 들어가 있습니다.

**Q. 명령어를 잘못 쳤어요.**
A. 걱정 마세요. 원본 파일은 그대로 남고, 결과는 새 파일로 저장됩니다. 다시 시도하면 됩니다.

**Q. 도장(직인) 이미지는 어떤 파일이어야 하나요?**
A. 흔히 쓰는 그림 파일(png, jpg, gif, bmp)이면 됩니다. 크기는 "2cm로 넣어 줘"처럼
말하면 비율을 지키며 맞춰 줍니다.

**Q. 개인정보(이름·연락처)를 채워도 안전한가요?**
A. 개인정보 양식은 값이 화면·기록에 남지 않는 특별한 방식으로 채우고,
끝나면 정보 파일을 안전하게 지우는 것까지 안내합니다.

## 8. 어려운 말 풀이 (용어 사전)

| 말 | 쉬운 뜻 |
| --- | --- |
| **HWPX (.hwpx)** | 요즘 한글 프로그램이 쓰는 한글 파일 종류. 두 번 클릭하면 한글로 열립니다. |
| **스킬(Skill)** | 앱에 끼워 넣는 '기능 꾸러미'. 한 번 등록하면 그 기능을 쓸 수 있습니다. |
| **재기안** | 있던 문서의 모양은 두고 **내용만 새로** 바꾸는 것. |
| **표준양식** | 우주항공청이 정한 보고서 기본 틀(표지·로고·여백). |
| **본문 위계 (□ ㅇ - ※)** | 큰 항목(□) 아래 세부(ㅇ), 그 아래(-), 참고(※)로 이어지는 글머리 단계. |
| **검증/검사** | 만든 보고서가 규정에 맞는지 자동으로 확인하는 일. |
| **표기법 검사** | 날짜(`2026. 7. 7.`)·시간(`14:30`)·숫자 쉼표(`12,500`) 같은 공문서 쓰기 규칙을 자동으로 확인하는 일. |
| **직인** | 기관 도장. 이 도구는 도장 **그림 파일**을 보고서의 정해진 자리에 넣어 줍니다. |
